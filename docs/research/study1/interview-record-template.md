# 访谈记录表 Interview Record Template

> **范围与伦理提示（务必先读）**
> 本表是 Study 1（第一阶段访谈）的**研究准备材料**，用于试访谈（pilot）。目前**尚未招募真实参与者、尚未开展真实老年人访谈、不含任何 PII（个人身份信息）**。
> - 本表**不记录**真实姓名、电话、住址、身份证号、具体单位名称等可识别信息，一律使用**参与者编号**（如 `P01`）。
> - **在开展任何真实访谈之前，必须先与导师确认伦理审批与知情同意流程，并取得参与者书面/口头知情同意。**
> - 安全底线：**不做医疗诊断、不给用药剂量建议**；涉及已故亲友（T09）时，系统与所有角色**绝不扮演逝者**；对高/中-高敏感话题**不强行深入、随时可停、由参与者主导**。

### 记录规范速记（供研究者）

- **不写 PII**：真名 → 编号；地名/单位 → 大类描述；引用 → 去标识后再录入。
- **编码一致性**：话题用 `T-code`、角色用 `R-code`、记忆用 `M-code`、线索组用 `C-code`，与话题卡 / 角色卡 / 引入脚本 / 记忆卡片 / 本记录表一一对应。
- **敏感度分级**：低＝可自由展开；中＝因人而异、留意情绪、可跳过；高＝敏感、不强行深入、随时可停、由参与者主导。
- **安全底线**：不诊断、不给用药剂量建议；T09 绝不扮演逝者、不追问细节。
- **参与者不是被试而是共创者**：以其舒适与主导为先，随时可停。
