# 话题卡 Topic Cards

> 用途：第一阶段（Study 1）半结构访谈 / 共创中的话题卡。用于了解老年参与者愿意聊什么、需要谨慎聊什么，以及每类话题适合什么样的 AI 陪伴角色。
>
> 范围诚实声明：这是研究筹备材料，用于试访（pilot interview）。**目前尚未招募真实参与者，也尚未开展真实老年访谈；材料中不含任何个人身份信息（PII）。** 正式访谈前，需与导师确认伦理审查与知情同意流程。
>
> 交叉一致性：本文件用 T 编码（T01–T12）引用话题。角色卡用 R 编码（R1–R8），记忆卡片用 M 编码（M1–M5），引入脚本用 C 编码（C1–C4）。同一套编码在话题卡 / 角色卡 / 引入脚本 / 记忆卡片 / 记录表之间通用，便于对齐。

---

## 一、怎么用这些卡片

**基本玩法（话题排序）**

把 12 张话题卡摊在桌上（或依次读给参与者听），请参与者把每张卡放进三堆之一：

- 🟢 **愿意聊** —— 想到就能讲，比较放松。
- 🟡 **看情况** —— 要看人、看当时的心情和气氛才愿意聊。
- 🔴 **不太想聊** —— 现在不太想碰，可能因为隐私、伤心或没兴趣。

排完之后，针对几张关键卡追问一句"为什么放这里"，并顺便问：**如果是 AI 陪您聊，哪些话题它可以主动提？哪些必须先问过您？**

**每张卡片包含四部分：**

| 栏目 | 说明 |
|---|---|
| 编号 | T-code，全套材料通用。 |
| 给参与者看的表述 | 可直接读给参与者听、或递给参与者看的一句话。用温和的日常口语，不含任何研究术语。 |
| 研究记录含义 | 只给研究者看。记录从这张卡能编码出什么（话题意愿、适配的关系角色 R 编码、敏感度信号等）。 |
| 敏感程度 | 低 / 中 / 高（含"不强行深入"提示）。 |

> 提示：**不要**把"研究记录含义"这一栏读给参与者听——那是给研究者做编码用的。递卡时只读"给参与者看的表述"。

## 二、敏感话题处理提醒（务必先读）

**三级敏感度刻度：**

- **低** —— 可自由展开，AI 或访谈者可较主动地邀请。
- **中** —— 因人而异；留意情绪，随时可跳过，先征求同意再深入。
- **高** —— 敏感话题，**不强行深入、随时可停、由参与者主导**节奏与深度。

**通用红线（对高 / 中-高话题尤其重要）：**

1. **不强行深入。** 参与者一旦露出不想继续的信号（沉默、转移话题、情绪波动），立即停下或转向，不追问细节。
2. **随时可停、可跳过。** 每张卡都可以说"这张我们跳过"。
3. **由参与者主导。** 敏感话题里，让参与者决定讲多深、讲哪部分。
4. **T09（已故亲友）绝不扮演逝者。** AI / 角色永远不模仿、不代言已故者，也不追问离世细节。
5. **T10（健康照护）不诊断、不给用药剂量建议。** 只做情绪陪伴与倾听，必要时建议联系医生 / 药师 / 家人。
6. **家庭矛盾（可能出现在 T03）** 视同中-高敏感处理：不强行深入。

如果访谈者拿不准某张卡该不该继续，默认**降速、确认、必要时跳过**。

## 三、话题卡 T01–T12

### 🟢 T01 · 年轻时的学习经历

- **给参与者看的表述：**
  "想跟您聊聊年轻时候读书、上学那会儿的事。您还记得念书时候的学校、老师，或者一起上学的同学吗？"
- **研究记录含义：** 话题意愿（低敏感话题的开场适配度）；适配角色 R1 同龄共鸣者 / R2 晚辈好奇者；是否适合作为破冰话题；关联引入脚本 C3（工作或学习经历）。
- **敏感程度：** 低。可自由展开，适合作为暖场话题。

---

### 🟢 T02 · 年轻时的工作经历

- **给参与者看的表述：**
  "您年轻时候是做什么工作的？当年上班、干活那会儿，有没有让您印象特别深的事？"
- **研究记录含义：** 话题意愿；经验讲述与传承意愿（是否想把经验讲给别人听）；适配角色 R1 同龄共鸣者 / R2 晚辈好奇者 / R3 中年传承者；关联引入脚本 C3。
- **敏感程度：** 低。一般可自由展开；若涉及下岗、失业等，留意情绪并放慢。

---

### 🟡 T03 · 家庭、孩子、教育经验

- **给参与者看的表述：**
  "想听您讲讲家里的事——比如孩子小时候，或者您在带孩子、教育孩子上的一些体会。您愿意聊多少都行。"
- **研究记录含义：** 话题意愿；家庭话题的开放度与边界；适配角色 R2 晚辈好奇者 / R3 中年传承者；可能触发 M4 敏感记忆（家庭矛盾）。
- **敏感程度：** 中（可能涉及家庭隐私或矛盾）。**不强行深入。** 若谈及家庭矛盾，按中-高敏感处理，留意情绪、可随时跳过、由参与者主导。

---

### 🟢 T04 · 老朋友、邻里、集体生活

- **给参与者看的表述：**
  "以前的老朋友、老邻居，或者一块儿过集体生活的日子，您还常想起来吗？那时候大家处得怎么样？"
- **研究记录含义：** 话题意愿；社交回忆的丰富度；适配角色 R1 同龄共鸣者；关联情感陪伴需求（与 T12 呼应）。
- **敏感程度：** 低。可自由展开；若牵出已故友人，转按 T09 红线处理。

---

### 🟢 T05 · 老照片或旧物件

- **给参与者看的表述：**
  "您家里有没有一些老照片，或者用了很多年、舍不得扔的老物件？它们背后有没有什么故事？"
- **研究记录含义：** 话题意愿；回忆触发物偏好（照片 / 旧物是否好用）；适配角色 R1 同龄共鸣者 / R6 回忆整理者；关联引入脚本 C1（老电视 / 旧物）、C2（老照片）。
- **敏感程度：** 低。可自由展开，是很好的回忆触发入口。

---

### 🟢 T06 · 老电影、戏曲、歌曲、地方文化

- **给参与者看的表述：**
  "您年轻时爱看什么电影、听什么戏或什么歌？咱们本地有没有什么老讲究、老风俗，是您特别有感情的？"
- **研究记录含义：** 话题意愿；文化 / 兴趣共同语言（可编码为 M2 兴趣记忆）；适配角色 R1 同龄共鸣者 / R5 主题陪伴者。
- **敏感程度：** 低。可自由展开，容易建立共同话题。

---

### 🟡 T07 · 重要人生选择

- **给参与者看的表述：**
  "回头看，有没有哪个重要的决定，让您觉得挺关键的？当时是怎么想、怎么定下来的？"
- **研究记录含义：** 话题意愿；反思深度与人生叙事意愿；适配角色 R3 中年传承者 / R4 长辈引导者；关联引入脚本无直接对应，属深入话题。
- **敏感程度：** 中。因人而异，留意情绪、可跳过。若引出遗憾，转按 T08 处理，**不强行深入**。

---

### 🔴 T08 · 人生遗憾或后悔

- **给参与者看的表述：**
  "有没有一件事，现在想起来还会觉得有点遗憾或后悔？如果不太想说，咱们随时可以停，或者换个话题。"
- **研究记录含义：** 话题意愿（高敏感话题的接受度）；情绪承接需求；适配角色 R4 长辈引导者 / R7 边界守护者；可能触发 M3 情感记忆 / M4 敏感记忆。
- **敏感程度：** 中-高。**不强行深入、随时可停、由参与者主导。** 只做温和的情绪支持，不评价、不追问细节。

---

### 🔴 T09 · 已故亲友

- **给参与者看的表述：**
  "如果您愿意，也可以跟我说说一位对您很重要、但已经不在了的亲人或朋友。您想讲多少都行，随时可以停下来。"
- **研究记录含义：** 话题意愿（最高敏感话题的接受度与边界）；哀伤信号；适配角色 R7 边界守护者（主导暂停 / 转向）；触发 M4 敏感记忆（默认不保存或强确认）。
- **敏感程度：** 高。**绝不扮演逝者，随时可停，不追问离世细节，不强行深入，由参与者完全主导。** 出现明显哀伤时，优先暂停与情绪安抚。

---

### 🔴 T10 · 身体健康和照护

- **给参与者看的表述：**
  "想问问您平时身体怎么样、生活上有没有需要人搭把手的地方。您方便说多少都行。"
- **研究记录含义：** 话题意愿；照护需求与陪伴期待；适配角色 R7 边界守护者；触发 M4 敏感记忆。**注意：这是访谈与陪伴话题，不是问诊。**
- **敏感程度：** 中-高。**不诊断、不给用药剂量建议、不强行深入。** 只倾听与情绪陪伴；涉及具体病情或用药，建议联系医生 / 药师 / 家人。

---

### 🟢 T11 · 新技术、新生活方式

- **给参与者看的表述：**
  "现在手机、网络、这些新东西越来越多，您平时用得怎么样？有没有觉得方便的，或者觉得别扭、不习惯的？"
- **研究记录含义：** 话题意愿；对新技术（含 AI 陪伴本身）的态度；适配角色 R2 晚辈好奇者 / R5 主题陪伴者；关联引入脚本 C4（新技术和新生活）。
- **敏感程度：** 低。可自由展开，也可自然过渡到对 AI 陪伴的看法。

---

### 🟡 T12 · 孤独、陪伴和日常心情

- **给参与者看的表述：**
  "平常一个人的时候，心里是什么感觉？有没有想找个人说说话、或者希望有人陪陪的时候？"
- **研究记录含义：** 话题意愿；孤独感与陪伴需求强度；适配角色 R4 长辈引导者 / R7 边界守护者；与 T04 社交回忆呼应。
- **敏感程度：** 中。留意情绪，**避免加深孤独感**（不渲染"只有我懂您"这类依赖式表达），可跳过、由参与者主导。
