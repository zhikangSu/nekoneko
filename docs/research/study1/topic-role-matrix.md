## 0. 这是假设，不是结论

**⚠️ 本矩阵是"待试访检验的研究假设"，不是已验证的发现。**

- 下表中的"推荐 / 辅助 / 不推荐"是研究者的**先验猜测**，用来给试访提供一个起点、方便临场判断，而**不是**规定老人"必须"如何反应。
- 真实老人的偏好可能与本表**完全相反**（例如某位老人最愿意让"晚辈好奇者 R2"陪着聊已故亲友 T09）。一旦试访中出现与本表冲突的信号，**以老人的真实反应为准**，并记入试访记录第 4/5 栏（哪些角色自然 / 哪些角色尴尬）。
- 本表的作用是帮助研究者**提出可被推翻的假设**，试访后应根据实际情况修订本文件，而不是拿本表去"证明"什么。

**贯穿全表的底线（安全 / 伦理）**

- **R8 不需要 AI 角色 永远是合法选项。** 任何话题下，老人都可以选择"不要任何角色陪，我自己讲/我不想讲"。R8 不是失败，是一种正常且需要被尊重的偏好。
- 敏感话题（T08 遗憾、T09 已故亲友、T10 健康照护、以及 T03 中的家庭矛盾）**从不主动追问、从不深挖**；老人可随时跳过或停止。
- **任何角色都绝不扮演逝者**（不模仿口吻、不代逝者说话、不"替他/她"回应）。这条对 T09 尤其关键。
- **不做医疗诊断、不给药物剂量建议**（T10）。角色只能倾听、共情、并在需要时建议联系家人/医生/药师。
- 对参与者**不暴露研究者术语**（多智能体 / 关系编排 / 角色间铺垫等）；这些词仅供研究者内部使用。

## 2. 话题–角色匹配矩阵（主表）

> 读法：**推荐角色**=试访时优先尝试；**辅助角色**=可作为备选或换口吻时使用；**不推荐角色**=在该话题上大概率会造成压力或越界，试访时先避开（除非老人主动要求）。
> R8 未逐行重复列出，因为它对**每一行都成立**——任何话题老人都可选"不需要 AI 角色"。

| 话题 | 推荐角色（primary） | 辅助角色（secondary） | 不推荐角色（avoid） | 原因 | 风险 |
|---|---|---|---|---|---|
| **T01 年轻时学习**（低） | R1 同龄共鸣者、R5 主题陪伴者 | R2 晚辈好奇者、R3 中年传承者 | R4 长辈引导者 | 低敏感、回忆愉快，同龄共鸣能勾出细节，晚辈好奇在这里显得真诚不冒犯 | R4 更权威的口吻可能让老人觉得"被考问/被评判成绩" |
| **T02 年轻时工作**（低） | R1 同龄共鸣者、R3 中年传承者 | R5 主题陪伴者、R2 晚辈好奇者 | R4 长辈引导者 | 工作经历有成就感，适合"传承"框架，同龄人能对上时代背景 | 若老人对下岗/单位变故敏感，R3"传承"口吻可能显得抬高；留意及时降温 |
| **T03 家庭/孩子/教育**（中，可能家庭矛盾） | R5 主题陪伴者、R7 边界守护者 | R1 同龄共鸣者 | R2 晚辈好奇者、R4 长辈引导者 | 话题温暖但**可能触及家庭矛盾**，需要能"陪着但不追问"的角色；R7 提前给出可以不聊的出口 | R2 追问"您和孩子关系怎么样"极易戳中痛处；R4 的评判口吻可能让老人觉得教育方式被指点 |
| **T04 老朋友/邻里/集体生活**（低） | R1 同龄共鸣者、R5 主题陪伴者 | R2 晚辈好奇者、R6 回忆整理者 | —（无明显不推荐） | 集体记忆、邻里往事最适合同代共鸣，气氛轻松 | 若聊到已故老友会滑向 T09，需及时切换到 T09 的守护策略 |
| **T05 老照片/旧物件**（低） | R5 主题陪伴者、R6 回忆整理者 | R1 同龄共鸣者、R2 晚辈好奇者 | —（无明显不推荐） | 有具体实物做锚点，"陪着看"和"帮您整理"都很自然，配合线索组 C1/C2 | 照片里可能出现逝者，一旦如此立即转入 T09 守护策略，绝不就逝者角色扮演 |
| **T06 老电影/戏曲/歌曲/地方文化**（低） | R1 同龄共鸣者、R5 主题陪伴者 | R2 晚辈好奇者、R6 回忆整理者 | —（无明显不推荐） | 文化记忆愉快、门槛低，同代人能共同哼唱、共享年代感 | 基本无风险；注意别把"考记忆"变成压力（如追问名字、年份） |
| **T07 重要人生选择**（中） | R6 回忆整理者、R5 主题陪伴者 | R1 同龄共鸣者、R3 中年传承者 | R2 晚辈好奇者、R4 长辈引导者 | 人生抉择需要被"梳理与尊重"，R6 帮老人自己看清脉络 | R2/R4 容易变成"当初为什么不……"式追问或评判，让老人感到被质疑选择 |
| **T08 人生遗憾/后悔**（中–高） | R7 边界守护者、R5 主题陪伴者 | R6 回忆整理者 | R2 晚辈好奇者、R3 中年传承者、R4 长辈引导者 | 高情绪、易脆弱，首要是**安全与出口**，只倾听不追问、不评判、不催"想开点" | R2 好奇追问=二次伤害；R3"把遗憾传下去"=把痛处对象化；R4 说教式安慰易显居高临下 |
| **T09 已故亲友**（高，**绝不扮演逝者**） | R7 边界守护者、R5 主题陪伴者 | R6 回忆整理者（仅在老人主动愿意留存时，谨慎） | R2 晚辈好奇者、R4 长辈引导者、R3 中年传承者 | 最高敏感度，核心是**守护 + 陪伴，不主动挖、不催、随时可停**；任何角色都不得扮演或代言逝者 | R2 追问细节=撕开伤口；R3/R4 试图"引导释怀/传承精神"可能显得功利或说教；**任何角色扮演逝者都是严重伦理越界，一律禁止** |
| **T10 身体健康/照护**（中–高，**不诊断/不给剂量**） | R7 边界守护者、R5 主题陪伴者 | R1 同龄共鸣者（共情"我也这把年纪"层面，不谈治疗） | R2 晚辈好奇者、R4 长辈引导者 | 涉及身体脆弱与照护焦虑，需守护式陪伴与安全出口；只倾听、只共情、必要时建议联系家人/医生/药师 | R2 追问病情=施压；R4 权威口吻易滑向"你应该吃什么药/该怎么治"——**绝不诊断、不给剂量**；任何"建议疗法"都属越界 |
| **T11 新技术/新生活**（低） | R2 晚辈好奇者、R5 主题陪伴者 | R3 中年传承者、R1 同龄共鸣者 | R4 长辈引导者 | 唯一"晚辈好奇"天然合适的话题：请老人讲对新事物的看法，晚辈请教式提问很自然，配合线索组 C4 | R4 居高临下地"教您用手机"易让老人觉得被当成落伍者、有挫败感 |
| **T12 孤独/陪伴/日常心情**（中） | R5 主题陪伴者、R7 边界守护者 | R1 同龄共鸣者 | R2 晚辈好奇者、R4 长辈引导者 | 情绪话题，重在被安静地陪着、被接住，而非被分析或被鼓劲 | R2 追问"您平时是不是很孤单"=贴标签、易触发防御；R4 说教"要多出去走走"显得不被理解；注意别写成加深依赖的话术 |

## 3. 敏感话题专用提醒（T08 / T09 / T10 / T03-家庭矛盾）

这四类话题在试访中**从不主动引导、从不深挖**，只在老人自己带出来时被动承接：

- **统一首选 R7 边界守护者**：先给出"可以不聊 / 随时停下"的出口，再决定要不要往下。
- **次选 R5 主题陪伴者**：安静地陪，不追问、不评判、不催"想开/放下"。
- **一律避开 R2 晚辈好奇者**（好奇=追问=施压）与 **R4 长辈引导者**（易变说教）；在 T08/T09 也避开 **R3 中年传承者**（把痛处"对象化/功利化"）。
- **T09 铁律**：任何角色**绝不扮演逝者**、不代逝者说话、不模仿其口吻。
- **T10 铁律**：**不诊断、不给药物剂量**；只能建议联系家人 / 医生 / 药师。
- **R6 回忆整理者**在 T08/T09 只能在**老人明确表示愿意留存**时、且以最谨慎方式介入；默认不主动提"要不要把这个存下来"。相关记忆多属 **M3 情感记忆（谨慎保存）** 或 **M4 敏感记忆（默认不保存、需强确认）**，处理请对照 `memory-card-examples.md`，选项固定为 [保存] / [修改后保存] / [不保存] / [以后不要再提]。

## 4. 与试访记录的对接

试访后请把本表相关观察填入 8 字段试访记录（`interview-record-template.md`）中，尤其是：

- **哪些角色自然** → 验证/推翻本表"推荐 + 辅助"列；
- **哪些角色尴尬** → 验证/推翻本表"不推荐"列；
- **哪些话题明显敏感** → 校准 §0 底线与 §3 专用提醒；
- **多角色短对话是否可接受** → 关系配合（线索组的 C 型"多角色短对话"）在不同话题上的可接受度，可能因话题敏感度不同而不同。

分析编码时，本表主要对应 Study 1 codebook 中的
**Relationship-role fit（角色契合）**、**Role rejection（角色拒绝）** 两类；
若某话题下老人反复选择 R8，则同时记入 **Control need（控制需求）**。
其余类别（Topic preference / Cue preference / Speaking pressure / Social presence / Cognitive load / Memory boundary / Ethical boundary）请对照 `study1-interview-guide-v2.md` 的完整 codebook。
