# Pilot Summary Template

> 用途：在完成 1–2 场**试访谈（pilot）**后 24 小时内填写的一页纸总结。
> pilot 的目的不是收集正式数据，而是检查材料是否好用：**哪些角色自然、哪些不自然、哪些话题不适合多角色 cue**，据此在正式 Study 1（8–12 人）前修订材料。
> 关联文件：`topic-cards.md` · `role-cards.md` · `conversation-cue-scripts.md` · `memory-card-examples.md` · `interview-record-template.md` · `study1-interview-guide-v2.md`。
> 编码复用同一套：话题 T01–T12 · 角色 R1–R8 · 记忆 M1–M5 · cue 组 C1–C4 · 编码表 10 类。
> 边界：pilot 不做录音、不记真名、不记 PII；敏感话题（T08/T09/T10/家庭矛盾）从不追问；角色从不扮演逝者；不诊断、不给药物剂量；正式访谈前须与导师确认伦理与知情同意。

## 6. 敏感边界是否守住（T08 / T09 / T10）

> 逐项确认。任何一项为「否」都必须在进入正式 Study 1 前修复。

- [ ] T08 遗憾 / T09 已故亲友 / T10 健康：**均未被追问**，长者说「不想聊」即停。
- [ ] 任何角色**都没有扮演逝者**（T09 绝不 role-play 逝者）。
- [ ] 涉及健康时**未做诊断、未给药物剂量**，只做倾听/陪伴。
- [ ] 家庭矛盾（T03）未被深挖，未站队评判。
- [ ] 全程无录音、无真名、无 PII；卡片/记录均用代号。
- [ ] 敏感话题**未使用多角色 cue**，仅用直接提问或单角色铺垫。
