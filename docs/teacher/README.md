# Teacher-facing research package

This folder contains materials that are suitable to share with the supervisor.

## Recommended teacher package

For the current research-preparation / Study 1 design stage, prioritize:

1. `relationship-aware-research-proposal.md`
2. `relationship-aware-interview-questionnaire.md`
3. `relationship-aware-opening-report.html`

The proposal and interview questionnaire are the two core files to send first.
The HTML file can be used as an opening-report / meeting deck.

## Internal source materials

The fuller working notes are kept separately under:

`docs/research/source-materials/`

Those files are useful for internal research preparation, literature/method
planning, and next-step execution, but they are not the default teacher-facing
package unless the supervisor asks for the full working process.
